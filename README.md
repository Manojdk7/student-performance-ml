# 🎓 Student Performance Prediction (Machine Learning Project)

## 📌 Overview
This is a beginner-friendly Machine Learning project that predicts student final scores based on study habits and academic factors.

The goal of this project is to understand how different features like study hours, attendance, and assignments affect student performance.

---

## 🚀 Features
- 📊 Data preprocessing using Pandas  
- 🤖 Model training using Linear Regression  
- 📈 Model evaluation (MAE, MSE, RMSE, R² Score)  
- 📉 Data visualization (Histogram & Scatter Plot)  
- 🔮 Predict student scores for new input  

---

## 🛠️ Tech Stack
- Python  
- Pandas  
- NumPy  
- Scikit-learn  
- Matplotlib
